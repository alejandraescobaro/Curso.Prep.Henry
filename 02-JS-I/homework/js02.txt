Variables: es la forma de almacenar el valor de algo para usarlo mas tarde. se usa la palabra clave Var y se asigna el valor usando el signo (=)

Strings: son bloques de texto. Cualquier texto entre comillas ("") es un string.

Funciones (argumentos, `return`): Una funcion es un conjunto de codigos que se agrupan para realizar una tarea concreta y se pueden reutilizar fácilmente.
Los argumentos: son las variables que necesitan las funciones, se agregan dentro de parentesis y si son multiples argumentos se separan por comas. 
return: Se usa para devolver los valores que se han calculado. Solo se puede devolver un valor cada vez que se ejecuta. Suele ser la ultima instruccion en las funciones que se necesita porque lo que se agregue despues de un return se ignora.

Declaraciones `if`: es una estructura de control de flujo usada para tomar decisiones. If es un condicional que sirve para realizar una u otra operacion en funcion de una expresion; es decir, es una forma en la que la funcion verifique si algo es "true" y ya sea ejecutando el codigo si sea asì, o si no lo es se avance. 

Valores booleanos (`true`, `false`): los valores booleanos son procedentes de la logica boole. es un concepto que alimenta el codigo binario. el codigo binario esencialmente significa que se tienen 2 opciones, activar o desactivar, 0 o 1, verdadero o falso. y en JavaScript se usan boolenos para significar verdadero o falso como "true" "false" respectivamente.
